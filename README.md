Aparri Coast Guard Web Dashboard for ESP32, GPS & GSM. 

The web-based dashboard allows authorized Coast Guard personnel to securely access real-time boat location data, monitor patrol routes, and coordinate operations through a centralized interface. By limiting system access to Coast Guard users only, the project ensures operational security, data confidentiality, and focused functionality

This system is designed to provide real-time boat tracking using LoRa, GSM, and ESP32 technologies, allowing efficient monitoring of coastal vessels and faster response during emergencies.
